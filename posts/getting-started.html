2026-06-17

欢迎！

欢迎来到 Furida！这篇文章将帮助你快速了解博客的功能和使用方法。

什么是 Furida？

Furida 是一个现代的个人博客平台，集成了 AI 聊天功能。 它完全免费，可以在 GitHub Pages 上免费托管，并通过 CDN 在中国大陆地区实现快速访问。

主要功能

  • 📝 博客文章 - 发布你的想法和经验
  • 💬 AI 聊天 - 与智能助手进行对话
  • 📱 响应式设计 - 在任何设备上完美显示
  • ⚡ 高性能 - 快速加载和流畅交互

如何使用

1. 浏览文章

点击导航栏的"文章"链接查看所有博客文章。 你可以点击任何文章的标题来阅读完整内容。

2. 使用 AI 聊天

点击导航栏的"AI 聊天"链接打开聊天界面。 输入你的问题或消息，点击"发送"按钮与 AI 助手交互。

试试这些问题：

  • 你好
  • 你是谁
  • 帮助

3. 本地运行

如果你想在自己的计算机上运行博客： 

cd furida
python -m http.server 8000

然后在浏览器中访问 http://localhost:8000

常见问题

博客可以免费使用吗？

是的，完全免费！使用 GitHub Pages 托管，无需任何费用。

可以在中国访问吗？

可以！我们使用了 CDN 加速和优化技术，确保在中国大陆地区的快速访问。

如何添加新文章？

在 posts/ 目录下创建新的 HTML 文件，然后在 index.html 或 posts.html 中添加链接。

如何集成我自己的 AI 模型？
