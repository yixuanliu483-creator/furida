关于 Furida

2026-06-17

项目介绍

Furida 是一个结合了个人博客和 AI 聊天功能的现代网站。 它完全免费、无需任何服务器成本，可以托管在 GitHub Pages 上， 并通过 CDN 在中国大陆地区实现高速访问。

核心特点

  • 📝 简洁的博客系统 - 使用 HTML 存储文章，无需数据库
  • 💬 AI 聊天集成 - 支持多种 AI 服务和自定义模型
  • ⚡ 快速加载 - 完全静态网站，优化了中国访问速度
  • 📱 响应式设计 - 完美适配手机、平板和桌面
  • 🔒 隐私保护 - 无需上传用户数据，完全由你掌控
  • 🚀 易于扩展 - 模块化代码，轻松添加新功能

技术栈

虽然项目名中有 Python，但实际上主要使用的是：

  • HTML5 - 页面结构
  • CSS3 - 样式和响应式设计
  • JavaScript (Vanilla) - 交互和聊天功能
  • GitHub Pages - 免费托管
  • jsDelivr CDN - 国内加速

未来可选集成 Python 后端（Flask/FastAPI）来实现更复杂的功能。

为什么选择 Furida？

相比其他博客平台和网站建设工具，Furida 提供了：

  • ✅ 完全免费 - 零成本运营
  • ✅ 中国大陆友好 - 快速访问
  • ✅ 易于定制 - 源代码完全开放
  • ✅ AI 友好 - 为集成自定义 AI 预留接口
  • ✅ 无服务器 - 无需维护后端

快速开始

开始使用 Furida 只需三步：

  1. 克隆或 fork 这个仓库
  2. 在 GitHub Pages 设置中启用此仓库
  3. 访问你的博客网址开始写作！
